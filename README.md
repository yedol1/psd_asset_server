# PSD ASSET UPLOAD FILE

Express로 구현된 Node.js 백엔드 어플리케이션 을 활용한 이미지 업로드 서버입니다.

### 패키지 명세

- 빌드 시 어플리케이션에 사용된 패키지를 설치하기 위해서는 `package.json`, `package-json.json`, `yarn.lock` 중 1개의 파일이 저장소에 반영되어 있어야합니다.

## ⌨️ 명령어

### Install

```bash
  npm ci
```

### Start

```bash
  npm start
```

## 🏷️ 환경변수

- `SESSION_SECRET`: 사용자 비밀번호
